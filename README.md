# Angular-Node-CURD-demo
 Angular + Node curd demo
 Using Angular Material

 # Install
 
 Frontend (Angular): 

 cd angular-material-demo<br>
 npm install<br>
 npm start<br>

 Backend (Node):

 cd node-js-server<br>
 npm install<br>
 npm start<br>
